# Ola Booking Analysis 🚖 | Excel, SQL, Power BI

## 📌 Project Overview
This project focuses on analyzing **103,025 Ola booking records** to uncover booking patterns, ride performance, customer behavior, and cancellation trends.

The project workflow includes:

- **Excel** for data cleaning and preprocessing  
- **SQL** for data analysis and querying  
- **Power BI** for interactive dashboard creation and business insights  

The objective of this analysis is to identify operational trends, monitor ride performance, and support **data-driven decision-making**.

---

## 🛠️ Tools & Technologies Used

- **Excel** – Data Cleaning & Preprocessing  
- **SQL (MySQL)** – Data Analysis & Querying  
- **Power BI** – Dashboard Development & Data Visualization  

---

## 📂 Dataset Information

- **Total Records:** 103,025 Booking Records  
- **Domain:** Ride Booking / Transportation Analytics  

The dataset includes information related to:

- Booking Status  
- Ride Volume  
- Vehicle Type  
- Cancellation Details  
- Booking Value  
- Customer & Driver Ratings  
- Payment Methods  

---

## 🔍 Project Workflow

### 1. Data Cleaning (Excel)
- Handled missing values and inconsistencies  
- Standardized dataset formatting  
- Cleaned and prepared structured data for analysis  

### 2. SQL Data Analysis
Performed SQL queries to analyze:

- Ride booking trends  
- Successful vs cancelled rides  
- Customer and driver cancellation behavior  
- Vehicle-wise performance  
- Revenue and booking value insights  
- Customer ratings and driver ratings  

### 3. Dashboard Development (Power BI)
Built an interactive dashboard featuring:

- KPI Cards  
- Booking Trends Analysis  
- Ride Cancellation Insights  
- Revenue Analysis  
- Vehicle Type Performance  
- Customer & Driver Ratings  
- Interactive Filters & Slicers  

---

## 📊 Key Insights

- **62% bookings were successfully completed**, while cancellations contributed significantly to ride loss.  
- **Cash and UPI** emerged as dominant payment methods.  
- Vehicle categories showed varying performance in booking value and ride distance.  
- Customer and driver cancellation reasons highlighted operational challenges.  
- Customer and driver ratings remained consistent across vehicle categories.
